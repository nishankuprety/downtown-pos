## Quick Start (Windows)
=======

**1. Install Dependencies:**
```cmd
npm install
cd frontend
npm install
cd ..
```
**2. Add Sample Menu:**
```cmd
node seed.js
```
**3. Start Application:**
```cmd
start.bat
```
**4. Open Browser:**
Go to http://localhost:3000
